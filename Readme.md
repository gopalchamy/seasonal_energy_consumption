#  Seasonal Energy Consumption Forecasting Using SARIMAX

 📌 Project Overview

This project implements advanced time series forecasting techniques to predict monthly electricity consumption for a power distribution company.

The objective is to capture strong seasonal patterns in energy demand while incorporating temperature as an external (exogenous) factor using SARIMA and SARIMAX models.

Accurate forecasting enables:
- Efficient energy distribution
- Prevention of power outages
- Optimized resource planning
- Improved infrastructure management

---

 🎯 Problem Statement

Electricity consumption exhibits strong seasonal patterns influenced by:

- Temperature variations
- Long-term demand growth
- Seasonal usage behavior
- External environmental factors

Traditional forecasting methods that ignore seasonality and exogenous variables result in inaccurate predictions and inefficient planning.

This project applies:

- SARIMA (Seasonal ARIMA)
- SARIMAX (Seasonal ARIMA with Exogenous Variables)

to improve forecast accuracy.
